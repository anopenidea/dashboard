About This Dashboard

This is a modern, responsive dashboard built with HTML, CSS, and JavaScript. It's designed to be simple, fast, and easy to customize.

Why I Built This

I use this every morning to start my day! Living in Houston, I needed one place to check the weather, know what to wear, and get motivated for the day ahead.

The dual temperature display (째F/째C) is especially helpful since I'm still learning to think in Fahrenheit. Having both units side-by-side makes it easier to understand the weather at a glance.

The smart clothing recommendations save me time every morning - no more guessing what to wear based on the forecast. And the daily motivational quotes help me start each day with the right mindset.

  • HTML5 - Structure and content
  • CSS3 - Styling and animations

  • Open-Meteo API - Free weather data
  • Air Quality API - US AQI data
  • GitHub Pages - Free hosting
